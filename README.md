# GradingSystem
Grading System Demo
1.0 2026/06/14
身份：教师, 管理员（有导出和删除数据权限）
需要实现的功能：
1. 进入时须输入用户名，密码。首次进入时需要修改密码。
2. 初始状态时，随机为用户分配考卷。
3. 评分时需要显示当前进度；当你退出系统时，需要保存当前进度。再次进入系统时，从第一份未提交评分的考卷开始评阅。若已经评阅完毕，则显示没有要评阅的试卷。
4. 要评阅的题目共有三个板块，满分分别为10+10+5，每个板块评分只能是整数。细则如下：	
内容	语言	调整项
A	9~10 9~10 5
B	7~8	7~8	4
C	5~6	5~6	3
D	3~4	3~4	2
E	0~2	0~2	1
要求：(i)内容和语言15分及以上，则调整项给3~5分；否则只能给0~3.
(ii)内容和语言必须在同一档或者相邻档给分，不可跨档。
如果给的分数不是整数，则弹出错误对话框：非法输入
如果不满足评分细则中的(i)和(ii)两点要求，则弹出错误对话框：错误给分。
